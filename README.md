
# 8086 Assembly Programs

## 📌 Overview
This repository contains a collection of **8086 Assembly Language programs** designed for learning low-level system programming, microprocessor architecture, and fundamental problem-solving using assembly language.

These programs were developed as part of academic coursework in Computer Science and Engineering, focusing on how a system operates at the hardware level.

---

## 🧠 Why Assembly Language?
Assembly language provides a deep understanding of how a computer works internally. It is essential for:
- Understanding CPU architecture and registers
- Learning memory management at low level
- Improving logical and problem-solving skills
- Building a strong foundation for **system programming and cybersecurity**

---

## 🔐 Cybersecurity Relevance
Although basic, assembly language plays an important role in cybersecurity and system security domains:

- Helps understand how **malware and exploits operate at low level**
- Useful for **reverse engineering and binary analysis**
- Builds foundation for **exploit development and vulnerability research**
- Improves understanding of **system calls, memory corruption, and stack behavior**

---

## 📂 Contents
This repository includes programs such as:

- Arithmetic operations (addition, subtraction, multiplication, division)
- Conditional logic and decision-making
- Loop-based programs
- Pattern generation
- Equation solving problems (e.g., linear expressions)
- Basic input/output handling in 8086 environment

---

## 🛠️ Tools Used
- EMU8086 / MASM / TASM
- Windows-based assembler environment

---

## 🎯 Learning Objectives
- Understand 8086 microprocessor architecture
- Develop low-level programming skills
- Strengthen logical thinking for system-level programming
- Build foundation for **cybersecurity and reverse engineering**

---

## 🚀 How to Run
1. Open any `.asm` file in EMU8086 or MASM/TASM
2. Assemble the program
3. Run and provide input (if required)
4. Observe output in console
